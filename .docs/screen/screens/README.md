# MyMeal App Screen Designs (SVG)

이 폴더는 MyMeal 앱의 UI/UX 화면 설계를 SVG 형식으로 포함합니다.

## 색상 테마

- **Primary**: `#0df214` (초록녹색)
- **Background**: `#f5f8f6` (연한 녹색 회색)
- **Surface**: `#ffffff` (흰색)
- **Text Primary**: `#111811` (진한 녹색 회색)
- **Text Secondary**: `#608a62` (보조 텍스트)

### 식사 유형 색상
- **아침**: `#FFB74D` (주황)
- **점심**: `#4DB6AC` (연하늘)
- **저녁**: `#7986CB` (보라연)
- **간식**: `#BA68C8` (연분홍)

### 상태 색상
- **좋음**: `#0df214` (초록)
- **보통**: `#FFC107` (노랑)
- **나쁨**: `#FF5252` (빨강)

## 사용 방법

각 SVG 파일은 375x812 (iPhone X/11/12 mini 기준) 크기로 설계되었습니다.

브라우저에서 직접 열거나 Figma, Sketch 등의 디자인 도구에서 가져와서 사용할 수 있습니다.
